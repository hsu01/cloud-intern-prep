# ☁️ Cloud Engineering Intern Prep Plan

This is my 6-week structured plan to prepare for my Cloud Engineering Internship.


# 🎯 Goal
By the end of this plan, I will:
- Understand AWS fundamentals (EC2, S3, IAM, VPC)
- Be comfortable using Linux CLI
- Use Terraform to provision infrastructure
- Understand Docker and basic Kubernetes
- Build and deploy a cloud project with CI/CD
- Feel confident starting my internship

---

# 🗺️ How to Use This Plan

- Study 2–4 hours per day
- Always do hands-on practice (not just watching)
- Log progress daily in GitHub
- Focus on understanding, not memorizing

---
